The last 24 hours were about agent control surfaces, not new flagship models. Claude Code pushed further into structured mid-run interaction through the Model Context Protocol (MCP) and better large-repo ergonomics, while OpenAI moved more production knobs into its Go SDK.

- **Claude Code adds structured MCP elicitation**: `v2.1.76` lets MCP servers request structured input mid-task and adds `Elicitation` hooks to intercept replies. Move auth, environment selection, and deploy parameters out of brittle prompts and into typed agent handoff flows.
- **Claude Code gets more controllable in monorepos**: the same release adds `worktree.sparsePaths`, `/effort`, `PostCompact`, and better background-agent result preservation. Restrict agent worktrees to touched directories and set explicit effort defaults before long repo-wide runs.
- **OpenAI Go SDK adds batch video, lazy tools, and a breaking voice change**: `v3.28.0` adds `/v1/videos` batch support, `defer_loading` for function tools, and changes `voice` from `string` to `string | {id}`. Update your Go types before bumping, defer heavyweight tool manifests until needed, and treat voices as first-class API objects.
